# Predicting-Paleoclimate-Using-Alkane-Biomarker-Proxies-and-Machine-Learning
GeoAI proof-of-concept that applies Random Forest machine learning to synthetic GC-MS alkane proxies data. Geochemical proxies including ACL, TAR, CPI, Paq, and OEP are used to predict vegetation type and paleoclimate conditions. This demonstrates the integration of organic geochemistry, Python, and ML for paleoenvironmental interpretation.

This project explores the application of Machine Learning in paleoenvironmental reconstruction using synthetic GC-MS-derived alkane biomarker data. A Random Forest classifier is trained on key geochemical proxies including Average Chain Length (ACL), Terrigenous-to-Aquatic Ratio (TAR), Carbon Preference Index (CPI), Aquatic Plant Proxy (Paq), and Odd-Even Preference (OEP) to predict vegetation type and paleoclimate conditions.

The workflow demonstrates how biomarker indicators commonly used in organic geochemistry can be integrated with Python-based machine learning techniques for environmental interpretation. Using a synthetic dataset generated to represent varying climatic and vegetation scenarios, the model learns relationships between alkane proxy signatures and corresponding paleoenvironmental classes.

This project serves as a proof-of-concept GeoAI application, showcasing data preprocessing, feature engineering, model training, classification, and prediction using Scikit-learn. It is intended for educational and exploratory purposes and does not represent a validated paleoclimate reconstruction model.
